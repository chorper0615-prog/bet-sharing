🚀 配置 Supabase 实时同步

需要运行一段 SQL 来启用实时数据同步功能。

步骤 1

点击下方按钮，打开 Supabase SQL 编辑器：

打开 Supabase SQL 编辑器

如果未登录，请先登录你的 Supabase 账号

步骤 2

复制以下 SQL 到编辑器中：

alter publication supabase_realtime add table if not exists bet_shares;
alter table public.bet_shares replica identity full;

步骤 3

按 Ctrl + Enter 或点击 "Run" 按钮运行 SQL。

✅ 完成后刷新游戏页面，同步功能即可工作！
